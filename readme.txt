Show a rss feed in a "bandeau" at the bottom of the browser.

== Description ==

	
First, news, you can now choose to have several streams and determine the speed of change in the flow of titles displayed. Optimization later, the style sheet is no longer included in the page but in a file devoted to hook Header engine Wordpress blog.

A small orange button with a down arrow has occurred, it can quickly return to the download page of this plugin.

To take account of new capacity, the interface configuration in the Control Panel the administrator has been heavily modified.

At the bottom of the admin interface, we can see the banner, it is exactly what gives the plug-in page of your blog. This is quite handy and can see immediately if the URL of a stream is not correct because nothing appears in this case.

== Installation ==


1. Upload files to the `/wp-content/plugins/llt/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
